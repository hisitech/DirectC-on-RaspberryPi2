# DirectC-on-RaspberryPi2
a FPGA(Actel/MicroSemi) programmer on RaspberryPi2,  Multchips support.
JTAG interface:
=================

    jtck GPIO-22;
    jtdi GPIO-23;
    jtms GPIO-24;
    jtdo GPIO-25;
